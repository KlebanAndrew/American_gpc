Хмарные сервисы

The Silicon Power Gateway предоставляет пользователю самодостаточные и полнофункциональные настройки и возможности отображения. Однако, начиная с монтажников и лизинговых компаний из соглашения о закупке электроэнергии в места часто нужно дистанционно отслеживать данные о производстве, Chilicon Power Gateway приходят подготовленные с Wi-Fi и по Ethernet 100 baset с периферийными устройствами для включения удаленного подключения к нашему облачному сервису.

Облачный сервис предоставляет исторические данные по добыче и информацию о состоянии для каждого зарегистрированного инвертора. Для производственных данных существуют несколько уровней избыточности, непосредственно начиная с флэш-памяти инвертора, для резервного копирования в шлюзе информации, полученной от инвертора для корпоративного уровня для хранения данных в облаке сервисного центра.
